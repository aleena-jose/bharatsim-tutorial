BharatSim Documentation
========================

Welcome to the BharatSim Tutorial Documentation.

BharatSim is a collaborative project between Ashoka University and Thoughtworks. The vision for BharatSim is to build a distributed, multi-scale **agent-based model simulation framework** for the scientific community. 

In this tutorial documentation, we look at examples in Epidemiology.
